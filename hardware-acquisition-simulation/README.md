# hardware-acquisition-simulation
Project Summary (English)

This project simulates the acquisition of hardware for three different computer systems based on specific requirements:

.Gaming Desktop Computer

.Enterprise Server for a company infrastructure

.High-performance Laptop for a CEO

The project includes hardware selection, cost estimation and technical justification for each system.

The original project specification is available in the /enunciado folder.
The final project documentation (PDF) is available in the /projeto folder.

Note: The full project documentation is written in Portuguese.

Resumo do Projeto (Português):


Este projeto simula a aquisição de hardware para três tipos diferentes de computadores:

Computador desktop para gaming

Servidor para infraestrutura empresarial

Portátil de alto desempenho para um CEO

O enunciado do projeto encontra-se na pasta /enunciado.

O documento final do projeto encontra-se na pasta /projeto.
